# Neighbourhood Skill Sharing Platform

## Description

This is a MERN stack web application that allows users to share and learn skills within their local community. Users can register, add skills they can teach, explore available skills, schedule learning sessions, and provide ratings and reviews.

---

## Features

- User Registration & Login
- Add Skill (Teach)
- Skill Categories (Tech, Cooking, Art, Fitness, Academic)
- Explore Skills
- Schedule Learning Sessions
- Rating & Review System
- Dashboard
- User Profile

---

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Other: Axios, JWT Authentication

---

skill-sharing-app/

## Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/skill-sharing-app.git

### 2. Backend setup

cd backend
npm install
npm start

### 3. Frontend setup

cd frontend
npm install
npm run dev

## API Base URL

http://localhost:8000/api

## Core Workflow

Register → Add Skill → Explore Skills → Schedule Session → Give Rating
